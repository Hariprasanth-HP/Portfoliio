# Hari Prasanth K - Portfolio

A high-quality, interactive personal portfolio website built with modern web technologies, inspired by the sleek design of Brittany Chiang's portfolio.

## 🚀 Tech Stack

- **React 19**: Latest React features for peak performance.
- **Tailwind CSS 4**: Next-gen utility-first CSS with native performance.
- **Shadcn/UI**: Beautifully designed components built on Radix UI.
- **Framer Motion**: Smooth, high-performance animations and layout transitions.
- **Biome**: Extremely fast toolchain for linting and formatting.
- **Husky**: Git hooks for ensuring code quality before commits.
- **Vite**: Ultra-fast development environment and build tool.

## ✨ Features

- **Interactive Spotlight**: A smooth mouse-following glow effect.
- **Responsive Layout**: Two-column layout on desktop, single-column on mobile.
- **Active Navigation**: Auto-scrolling and active section detection.
- **Modern Dark Theme**: A sophisticated "Sophisticated Dark" aesthetic.
- **Smooth Scrolling**: Optimized scroll behavior for internal navigation.

## 🛠️ Getting Started

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Linting and Formatting

```bash
pnpm check  # Runs Biome check and applies fixes
pnpm lint   # Runs Biome linter
pnpm format # Runs Biome formatter
```

### Build

```bash
pnpm build
```

## 🚢 Deployment

The project includes a **GitHub Actions** workflow (`.github/workflows/deploy.yml`) ready for deployment to **GitHub Pages**.

To deploy:
1. Push your code to the `main` branch on GitHub.
2. Ensure GitHub Pages is configured to use GitHub Actions as the source in your repository settings.
